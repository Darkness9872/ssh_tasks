### Кейс 2: Подключение к удалённому серверу  
- Подключение по SSH к виртуальной машине
- IP: `192.168.0.249`
  
  Выполненные команды для достижения результата:
  ```bash
  # Узнаём IP виртуальной машины
  ip addr show
  
  # Подключаемся с хоста
  ssh ubuntu@192.168.0.249

### Кейс 3: Копирование файлов через SCP
- Файл `test_scp.txt` скопирован на удалённый сервер
- Скорость: `19.8KB/s`
  
  Выполненные команды для достижения результата:
  ```bash
  # Создаём файл на хосте
  echo "This is test file for SCP" > test_scp.txt
  
  # Копируем на виртуальную машину
  scp test_scp.txt ubuntu@192.168.0.249:~/

  # Копируем обратно
  scp ubuntu@192.168.0.249:~/test_scp.txt ./scp_test_backup.txt

  # Смотрим что пришло
  cat scp_test_backup.txt

### Кейс 4: Работа с SSH-ключами
- Настроен беспарольный доступ по ключам
- Подключение без ввода пароля
  
  Выполненные команды для достижения результата:
  ```bash
  # Генерируем ключи на хосте
  ssh-keygen -t rsa
  
  # Копируем ключ на виртуальную машину
  ssh-copy-id ubuntu@192.168.0.249
  
  # Подключаемся без пароля (после первого входа)
  ssh ubuntu@192.168.0.249
